Linux or Cygwin is the recommended platforms for compilation.
If the detected operating system IS NOT Linux or Cygwin, the program will not compile correctly.

If the detected operating system IS Linux or Cygwin, run 'make' in the current working directory.
An executable 'Bayes' will be created. Run './Bayes' to display results of Bayes Classifier.

This will create output streams to all sets of generated data in the ./data folder.
